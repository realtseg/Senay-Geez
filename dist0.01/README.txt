Text Substituter - Portable Version 
=============================================== 
 
USAGE INSTRUCTIONS: 
1. Run "Start Text Substituter.bat" to start the program 
2. Press PAGE UP key to toggle text substitution ON/OFF 
3. A small overlay will appear near your taskbar showing status 
4. Blue icon = RUNNING, White icon = STOPPED 
 
FEATURES: 
- Type Amharic characters using Latin keyboard 
- Works in any application 
- No installation required 
- Portable - can run from USB drive

Edit config.csv to add your own text substitutions
